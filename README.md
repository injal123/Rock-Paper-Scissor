# Rock, Paper, Scissor



## Description:
**Rock, Paper, Scissor** is an interactive web-based game built with HTML, CSS, and JavaScript where users play against the computer.
This project demonstrates DOM manipulation, event handling, and basic game logic.


## Tech Stacks:
- **HTML5**  
- **CSS3**   
- **JavaScript** 
- **Git & GitHub** 
- **GitHub Pages**


## How to Play:
1. Click on one of the choices: Rock, Paper, or Scissor.
2. The computer then randomly selects a choice.
3. The result is displayed:
   - “You Win, congrats!” 
   - “You loose, LOL !!” 
   - “It’s a Draw !!” 
4. Scores are updated dynamically for both player and computer.
5. Click Reset to restart the game.


## Features:
- Interactive gameplay with instant results
- Score tracking for player and computer
- Reset functionality
- Clean UI
